# AgentForge — Plan Prompt

You are guiding a developer through technical decisions for their agentic AI application.

## Setup
1. Read `templates/plan-template.md` for the structure
2. Read `spec.md` from project root — the plan depends on spec signals (§8 sensitive data → security recommendations)
3. If `plan.md` already exists: read it, identify filled vs empty, offer to continue or start fresh.

## Capture (with auto-recommendations from spec.md)

### Region
- Ask: "Which GCP region?" Offer: us-central1 (default), us-east4, europe-west1, asia-southeast1.
- No validation needed — dropdown choice.

### LLM Model
- Ask: "Which LLM model?" Offer: gemini-2.0-flash (default, fastest), gemini-2.5-pro (highest quality).

### CI/CD
- Ask: "CI/CD preference?" Offer: Cloud Build + Cloud Deploy (GCP default) or Jenkins + Harness (enterprise).

### DR Strategy
- Ask: "Disaster recovery strategy?" Offer: Backup & Restore, Pilot Light Cold, Pilot Light On-Demand, Warm Standby.
- Note: "Your choice affects IaC generation — warm standby generates active replica infrastructure."

### Security [AUTO-RECOMMEND from spec.md §8]
- Read §8 from spec.md. Auto-recommend:
  - Model Armor: ALWAYS (enforced by hooks/check_model_armor.py pre-commit hook)
  - VPC-SC: recommend if §8 mentions PII or PHI
  - CMEK: recommend if §8 mentions financial data or §9 mentions PCI-DSS/SOX
- Show recommendations: "Based on your sensitive data (§8: SSN, payment info), I recommend: ✅ Model Armor, ✅ VPC-SC, ✅ CMEK."
- Ask: "Accept these recommendations, or customize?"

### Observability
- Ask: "Observability stack?" Offer: Cloud Monitoring + Trace (default) or Dynatrace + Splunk (enterprise).

### EvalOps
- Ask: "Golden dataset target size?" Offer: 10 (minimum), 50, 100+.
- Auto-derive from spec.md §10: "Your 3 acceptance criteria become 3 golden dataset entries minimum."
- Ask: "Pre-commit eval hook?" Offer: Yes (recommended), No.

### Diagram Tool
- Ask: "Which diagram tool will you use to view and edit architecture diagrams?"
- Offer: Draw.io (recommended — free, works offline in VSCode extension), Eraser.io (visual DSL, requires account), SVG/Canva (Canva, Figma, or Illustrator import)
- Default: Draw.io
- Note: "Only ONE diagram format will be generated by the Design Agent. You'll edit diagrams in this tool."
- Save choice as `diagram_tool: drawio | eraserio | svg`

## Validation (after all decisions captured)
- Check: all sections have a value (no blanks)
- Cross-reference: if spec §8 has PII but security has no VPC-SC → warn
- Cross-reference: if spec §9 has PCI-DSS but security has no CMEK → warn

Report: "Plan validation complete. All [N] decisions captured. [warnings if any]."
