# Deploy {{app_name}} to Google Cloud

> Generated by `/implement`. **Docs only** ‚Äî run these commands yourself; `/implement` does not deploy.

**APP_DIR:** `apps/{{app_slug}}/`  
**Primary region:** {{primary_region}}  
**Auth mechanism:** {{auth_mechanism}} (from `auth-config.yml`)  
**Official ADK reference:** [Deploy to Cloud Run](https://adk.dev/deploy/cloud-run/index.md)

## Prerequisites

1. **Google Cloud project** with billing enabled  
2. **APIs enabled:** Cloud Run, Cloud Build, Artifact Registry, Secret Manager, Vertex AI / Agent Platform (if using Vertex auth)  
3. **gcloud CLI** authenticated:
   ```bash
   gcloud auth login
   gcloud auth application-default login   # if auth-config mechanism is vertex_adc
   gcloud config set project {{google_cloud_project}}
   ```
4. **Python env** (local smoke before deploy):
   ```bash
   cd apps/{{app_slug}}
   pip install -e .
   PYTHONPATH=. python -c "from app.agent import root_agent; print(root_agent.name)"
   ```

## Path A ‚Äî ADK CLI (recommended for Python ADK agents)

Per [ADK Cloud Run deploy](https://adk.dev/deploy/cloud-run/index.md):

```bash
export GOOGLE_CLOUD_PROJECT="{{google_cloud_project}}"
export GOOGLE_CLOUD_LOCATION="{{primary_region}}"
export GOOGLE_GENAI_USE_VERTEXAI=TRUE   # or GOOGLE_GENAI_USE_ENTERPRISE per your ADK version
export AGENT_PATH="./app"               # directory containing agent.py + root_agent
export SERVICE_NAME="{{app_slug}}"

# From APP_DIR ‚Äî agent exports root_agent in app/agent.py
adk deploy cloud_run "$AGENT_PATH" \
  --project="$GOOGLE_CLOUD_PROJECT" \
  --region="$GOOGLE_CLOUD_LOCATION" \
  --service_name="$SERVICE_NAME"
```

**Auth notes (match `auth-config.yml`):**

| Mechanism | Cloud Run setup |
|-----------|-----------------|
| `ai_studio_api_key` | Store `GOOGLE_API_KEY` in Secret Manager; mount as env var |
| `vertex_adc` / `service_account` | Attach runtime SA with `roles/aiplatform.user`; use Workload Identity ‚Äî no key file on Cloud Run |
| `express_mode` | `GOOGLE_GENAI_API_KEY` via Secret Manager |

Grant Cloud Build permission (first deploy):

```bash
gcloud projects add-iam-policy-binding "$GOOGLE_CLOUD_PROJECT" \
  --member="serviceAccount:$(gcloud projects describe $GOOGLE_CLOUD_PROJECT --format='value(projectNumber)')-compute@developer.gserviceaccount.com" \
  --role="roles/cloudbuild.builds.builder"
```

## Path B ‚Äî Docker + `gcloud run deploy`

If `Dockerfile` exists in APP_DIR:

```bash
cd apps/{{app_slug}}
export PROJECT="{{google_cloud_project}}"
export REGION="{{primary_region}}"
export IMAGE="gcr.io/$PROJECT/{{app_slug}}:latest"

gcloud builds submit --tag "$IMAGE"
gcloud run deploy {{app_slug}} \
  --image "$IMAGE" \
  --region "$REGION" \
  --platform managed \
  --allow-unauthenticated=false \
  --service-account "{{runtime_service_account}}" \
  --set-env-vars "GOOGLE_CLOUD_PROJECT=$PROJECT,GOOGLE_GENAI_USE_VERTEXAI=TRUE,GOOGLE_CLOUD_LOCATION=$REGION"
```

Wire secrets (example):

```bash
gcloud run services update {{app_slug}} --region "$REGION" \
  --update-secrets=GOOGLE_API_KEY=GOOGLE_API_KEY:latest
```

## Path C ‚Äî Terraform + CI/CD (production)

1. **Apply infrastructure** (generated under `deployment/terraform/`):
   ```bash
   cd deployment/terraform
   terraform init
   terraform plan -var-file=env/nonprod.tfvars
   terraform apply -var-file=env/nonprod.tfvars
   ```
2. **CI/CD pipeline** ‚Äî push to trigger `ci-cd/cloudbuild.yaml` ‚Üí Artifact Registry ‚Üí Cloud Deploy:
   ```bash
   gcloud builds submit --config=ci-cd/cloudbuild.yaml ..
   ```
3. **Agent Gateway / Identity** ‚Äî routes and per-agent tokens from Terraform outputs (`gateway_url`, agent SA emails).

## Post-deploy verification

```bash
# Service URL
gcloud run services describe {{app_slug}} --region {{primary_region}} --format='value(status.url)'

# Health (if /health exposed)
curl -s "$(gcloud run services describe {{app_slug}} --region {{primary_region}} --format='value(status.url)')/health"

# Logs
gcloud logging read "resource.type=cloud_run_revision AND resource.labels.service_name={{app_slug}}" --limit 20
```
